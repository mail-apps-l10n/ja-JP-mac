Internet Explorer ユーザのために

Microsoft® Internet Explorer をお使いなら、簡単に &brandFullName; を使いはじめることができます。

注意: この情報は Windows と Mac OS 用の &brandFullName; 向けです。

IE のお気に入りと設定について

&brandShortName; をはじめてインストールする場合は、お気に入りや Cookie、保存されたパスワード、その他のデータを含む既存の設定が Internet Explorer から移行されます。これにより、&brandShortName; をあなたの希望に添うようにカスタマイズする手間を減らします。

&brandShortName; の機能

  • タブブラウズ: 訪問しようとする各サイトを別のブラウザウインドウで開く代わりに、 複数のサイトを同じウインドウ内のタブに開くことができます。 また、タブのグループをホームページに設定することもできます。 詳しくは タブブラウズ をご覧ください。
  • ポップアップウインドウコントロール: 迷惑なポップアップとポップアンダーウインドウを抑止または許可します。
  • Cookie マネージャ: Cookie を受け入れるときの &brandShortName; の挙動を変更します。
  • ダウンロードマネージャ: 混乱を最小限にするために ファイルのダウンロードを 1 つのウインドウにまとめます。 ダウンロードごとに多数の不要なウインドウを使用することなく、 すべてのダウンロード状況を知ることができます。
  • 画像マネージャ: 特定の Web サイトやすべてのサイトからの画像の読み込みを無効にすることができます。 これは、Web サイトからの読み込みにかかる時間を短縮したいときに役立ちます。

キーボードショートカット

Microsoft Internet Explorer と &brandShortName; が同じショートカットキーの多くを共有していることに気づくでしょう。 すべてのショートカットキーの一覧は、&brandShortName; キーボードショートカットの一覧 をご覧ください。
