&brandFullName; を使う

&brandFullName; へようこそ！ &brandShortName; は Web ページを訪れ、 探索するためのインターネットブラウザです。

このセクションでは:
  • Web ページをナビゲートする
  • 検索する
  • ページをコピー、保存、プリントする
  • スピードと効率を改善する

Web ページをナビゲートする

ホームページを表示する

&brandShortName; を起動すると、あなたのホームページを表示します。 デフォルトでは &brandShortName; のホームページが表示されます。

ヒント:

  • 画面上に、より多くの Web コンテンツを表示するにはフルスクリーンを利用してください。 フルスクリーンは、&brandShortName; のツールバーを 1 つの小さなツールバーに凝縮します。 フルスクリーンを有効にするには、表示 > フルスクリーン を選択するか F11 を押下するだけです。
  • あなたのホームページにすばやく移動するには、&altKey;+Home を押下してください。

別のページに移動する

ロケーションバーにインターネットアドレスまたは URL を入力すると、新しい Web ページに移動します。通常、URL は "http://" で始まり、"http://mozilla.jp" のようにアドレスを識別する名前が 1 つ以上続きます。

  1. ロケーションバーをクリックし、そこにある URL を選択します。
  2. 訪れるページの URL を入力します。あなたの入力した URL がロケーションバーにあるテキストと置き換えられます。
  3. &enterKey; を押下してください。

ヒント: ロケーションバーの URL をすばやく選択するには、 &accelKey;+L を押下してください。

URL がわからない？ 訪れたいページを特定できるもの(例えば名前など) をロケーションバーに入力して &enterKey; を押してみてください。その用語による Yahoo! JAPAN の検索結果が表示されます。

リンクをクリックする

大抵の Web ページには、クリックして他のページへ移動するためのリンクがあります。

  1. マウスポインタが指し示す手の形に変わるところまでマウスを移動してください。 ポインタがリンクの上にある場合は常にこうなります。 多くのリンクは下線付きのテキストですが、ボタンや絵などのリンクもあります。
  2. リンクを一度だけクリックしてください。ネットワークがリンク先のページを探している間、 ステータスメッセージがウインドウの下部に表示されます。

前に訪れたページへ戻る

ページを再び訪れるにはいくつかの方法があります:

  • 1 つ前、または次のページへ移動するには 戻る または 進む ボタンをクリックします。
  • 1 つ以上前、または次のページへ移動するには、戻る や 進む ボタン上の小さな三角形をクリックします。 最近訪れたページの一覧が表示されるので、戻りたいページを選択してください。
  • ロケーションバーに入力した URL の一覧を参照するには、右端の下向き矢印をクリックします。 一覧から表示するページを選択してください。

  • 現在のセッション中に訪れたページから選択するには、履歴 メニューを開いて メニュー下部の一覧を使用してください。
  • 過去のそれぞれのセッション中に訪れたページから選択するには、履歴 メニューを開いて サイドバーに表示 を選択してください。 履歴サイドバーが表示されます。履歴サイドバーはフォルダの一覧を表示します。 フォルダをクリックすると、サブフォルダまたは Web ページのタイトルが表示されます。 見たい Web ページのタイトルをクリックしてください。

読み込みの中止と更新

ページの読み込みがとても遅く、ページを表示したくないときは 中止 ボタンをクリックしてください。

現在のページを再び読み込んで最新の状態にするには、更新 ボタンをクリックするか &accelKey;+R を押下してください。

タブブラウズ

一度に複数の Web ページを訪れる場合は、 タブブラウズを利用して速く簡単に Web をナビゲートすることができます。

タブブラウズは、1 つのウインドウ内に各々の Web ページを表示するタブを開きます。 複数の異なる Web ページを訪れるために、複数のウインドウを開く必要はありません。 これは、デスクトップのスペースを解放します。別のウインドウに切り替えることなく、 1 つのウインドウで Web ページを開いたり閉じたり、再び読み込むことができます。

詳しくは タブブラウズ をご覧ください。

サイドバーを使用する

サイドバーは、画面左側にブックマークや履歴を表示するための領域です。 同様に、拡張機能が新しい方法を追加するためにサイドバーを利用することがあります。

サイドバーに項目を表示するには 表示 > サイドバー を選択してください。そこからサイドバータブに表示するものを選択することができます。

検索する

Web を検索する

特定のトピックでの Web ページの検索は、いくつかの単語を検索バーに入力するだけで簡単に行えます:

例えば、レッサーパンダのいる動物園についての情報を検索したいときは:

  1. 検索バーをクリックします。
  2. "動物園 レッサーパンダ" と入力してください。 入力すると、現在の検索バーのテキストを置き換えます。
  3. &enterKey; を押下して検索します。

"動物園 レッサーパンダ" の検索結果がウインドウに表示されます。

検索エンジンを選択する

検索エンジンのアイコンをクリックして、他の検索エンジンに切り替えることができます。 Google など一部の検索エンジンは Web 全体を検索しますが、他に Amazon.co.jp などは特定のサイト内を検索します。

検索バーの管理

検索エンジンを追加、並べ替え、削除、初期設定に戻すには、検索エンジンのアイコンをクリックして 検索バーの管理... を選択してください。 検索エンジンを一覧内で移動、削除するには、検索エンジンを選択して適切なボタンをクリックします。 検索エンジンを追加... のリンクをクリックすると、新しい検索エンジンをインストールすることができます。

Web ページ内で選択した単語を Web 検索する

&brandShortName; は Web ページ内で選択した単語を Web 検索することができます:

  1. Web ページ内の単語を選択 (強調表示) します。
  2. 右クリック&ctrlKey; を押下しながらマウスボタンをクリックし、(検索エンジン) で検索: "(選択した単語)" を選択してください。

選択された単語の検索には現在選択されている検索エンジンが使用され、新規タブに検索結果が表示されます。

ページ内を検索する

現在表示されているページ内のテキストを検索するには:

  1. &accelKey;+F を押下するか、編集 > このページの検索... を選択すると、&brandShortName; の下部に検索ツールバーが表示されます。
  2. 検索するテキストを入力します。検索ボックスに何か入力すると同時に、自動的に検索が始まります。
  3. 検索ツールバーには以下の選択肢があります:
    • 次を検索: 現在のカーソル位置より下にあるページ内のテキストを検索します。
    • 前を検索: 現在のカーソル位置より上にあるページ内のテキストを検索します。
    • すべて強調表示: 現在のページ内にある検索文字列を強調します。
    • 大文字/小文字を区別: 検索する文字列の大文字と小文字を区別します。

同じ単語やフレーズを再検索するには、F3 を押下するか、編集 > 次を検索 を選択します。

ヒント: 検索ツールバーの クイック検索 モードを有効にするには、 &pref.menuPath; の 詳細 パネルの 一般 タブ内にある、キー入力時に検索を開始する &pref.singular;を有効にしてください。これを有効にすると、何かを入力すると同時に クイック検索 ツールバーが表示され、自動的に検索を開始します。クイック検索 ツールバーは、通常の検索ツールバーとは異なり、数秒間そのままにしておくと自動的に隠れます。

ページをコピー、保存、プリントする

ページの一部をコピーする

ページ内のテキストをコピーするには:

  1. テキストを選択します。
  2. メニューバーから 編集 > コピー を選択してください。

他のプログラムにテキストを貼り付けることができます。

ページ内のリンク (URL) または画像のリンクをコピーするには:

  1. リンクまたは画像の上にポインタを移動します。
  2. リンクまたは画像の上で 右クリック&ctrlKey; を押下しながらクリック してコンテキストメニューを表示します。
  3. リンクの URL をコピー、または 画像の URL をコピー を選択してください。 画像がリンクのときはメニュー項目のどちらかを選択してください。

他のプログラム、または &brandShortName; のロケーションバーにリンクを貼り付けることができます。

ページの一部、またはすべてを保存する

ページ内のすべてを保存するには:

  1. ファイル > 別名でページを保存 を選択してください。別名で保存ダイアログボックスを表示します。
  2. ページを保存する場所を指定してください。
  3. 保存する形式を指定してください:
    • Web ページ, 完全: 画像も加えて、すべての Web ページを保存します。 この選択肢は画像を元の状態で表示しますが、元の HTML のリンク構造は維持しません。 Web ページ全体を表示するのに必要な画像や他のファイルを保存するために、 新規ディレクトリが作成されます。
    • Web ページ, HTML のみ: 画像を除いて元のページを保存します。 この選択肢は元の HTML のリンク構造を維持します。
    • テキストファイル: 元のページをテキストファイルとして保存します。 この選択肢は元の HTML リンク構造を維持しませんが、 テキストエディタで Web ページのテキスト版を表示することができます。
  4. ファイル名を入力して 保存 をクリックしてください。

Web ページのフレームを保存するには:

  1. フレーム内にマウスポインタを移動します。
  2. フレームを &ctrlKey; を押下しながらクリック右クリックしてコンテキストメニューを表示します。
  3. このフレーム > 別名でフレームを保存 を選択してください。 保存ダイアログを表示します。
  4. ページを保存する場所を指定してください。
  5. 保存する形式を指定してください。
  6. ファイル名を入力して 保存 をクリックしてください。

ハードドライブにファイルを保存しておくと、 インターネットに接続していなくてもページを見ることができます。

ページ内の画像を保存するには:

  1. 画像の上にマウスポインタを移動します。
  2. 画像を &ctrlKey; を押下しながらクリック右クリックしてコンテキストメニューを表示します。
  3. 別名で画像を保存 を選択してください。 画像保存ダイアログを表示します。
  4. 画像を保存する場所を指定してください。
  5. ファイル名を入力して 保存 をクリックしてください。

ページを表示せずに保存するには (表示するための形式ではないページを取り出すのに役立ちます):

  1. ページへのリンクの上にマウスポインタを移動します。
  2. リンクを &ctrlKey; を押下しながらクリック右クリックしてコンテキストメニューを表示します。
  3. 別名でリンク先を保存 を選択してください。 保存ダイアログを表示します。
  4. ページを保存する場所を指定してください。
  5. ファイル名を入力して 保存 をクリックしてください。

重要: 一部のリンクは、クリックすると自動的にファイルをダウンロードしてあなたのハードディスクに保存します。 このようなリンクの URL は "ftp" で始まるか "au" や "mpeg" などのファイル形式の拡張子で終わります。 これらのリンクは、ソフトウェアやサウンド、または映像ファイルをサポートする ヘルパーアプリケーションを起動するかもしれません。

ヒント: 画像をデスクトップの背景として設定するには、画像を右クリック&ctrlKey; を押下しながらマウスボタンをクリックし、コンテキストメニューから デスクトップの背景に設定... を選択してください。

ページをプリントする

現在のページをプリントするには:

  • ファイル > プリント を選択してください。

選択されたテキストをプリントするには:

  1. 現在のページの内テキストを選択してください。
  2. ファイル > プリント を選択してください。 プリントダイアログを表示します。
  3. プリント範囲 セクションの 選択した部分 をクリックします。

プリントされるページの内容の配置は、画面上のウインドウサイズではなく、Web ページ作者とプリント後のページサイズによって決定されます。 テキストは用紙サイズに合うように折り返され、画像は再配置されます。

プリントプレビューを使用する

プリントプレビューを使用すると、プリント前にそのページがどのようにプリントされるかを簡単に確認することができます:

  • ファイル > プリントプレビュー を選択してください。

プリントプレビューでは、プリントするページに対して次のようなことができます:

  • 各ページのプレビューを表示する: 次のページ または 前のページ , 最初のページ , 最後のページ ボタンをクリックすると各ページに移動します。
  • ページの縮尺(サイズ) を変更する: 拡大／縮小 ドロップダウンメニューで “50%” を選択すると 各ページを元の半分のサイズにします。“用紙に合わせて縮小” を選択すると ページサイズを自動的に用紙幅に合わせます。また “ユーザ設定” を選択すると 比率を指定することができます。
  • ページの方向を変更する: 縦 を選択するとページ幅の狭い方が上になります。 横 を選択するとページ幅の広い方が上になります。
  • ページの設定をする: プリントするページをさらにカスタマイズするには ページ設定 をクリックしてください。
  • プリントする: プリント をクリックするとページをプリントします。

注意: Mac OS と Linux では、プリントプレビュー機能が異なるか利用することができません。

ページ設定を使用する

注意: Mac OS と Linux では、プリントプレビュー機能が異なるか利用することができません。

ページ設定を使用してプリントするページをカスタマイズすることができます:

  • ファイル > ページ設定 を選択してください。

ページ設定でプリントしたいページの設定を次のように変更することができます:

  • 書式とオプション: プリント方向と縮尺、その他のオプションを選択してください
    • プリント方向:
      • 縦: を選択すると通常の配置になり、ページ幅の狭い方が上になります。
      • 横: を選択すると横向きの配置になり、ページ幅の広い方が上になります。
    • 拡大／縮小: 元のサイズに対する比率を入力します。 例えば、ページサイズを元の半分にするには "50" と入力してください。
      • ページ幅に合わせて縮める: これを選択すると、ページサイズを自動的に変更して用紙幅に合わせます。
    • オプション:
      • 背景色と背景画像もプリント: これを選択すると、背景の画像とカラーをプリントします。 選択されていないときは手前の画像とカラーのみがプリントされます。
  • 余白とヘッダ/フッタ: 余白とヘッダ、フッタを設定するにはこのタブをクリックしてください:
    • 余白:
      • 上、下、左、右: 決められた単位で上、下、左、右の余白を入力してください。
    • ヘッダとフッタ: 各ドロップダウンリストはヘッダとフッタのそれぞれの位置を表しています。 上の行はヘッダ部の左、中央、右に対応しています。 下の行はフッタ部の左、中央、右に対応しています。 各ドロップダウンリストで、次のオプションから 1 つを選択してください:
      • --なし--: この位置に何も表示しません。
      • タイトル: Web ページのタイトルを表示します。
      • URL: Web ページの URL を表示します。 (URL は通常、"http://" で始まります。)
      • 日付/時間: Web ページがプリントされたときの日付と時間を表示します。
      • ページ #: 各ページのページ番号を表示します。
      • ページ # / #: 全ページ数を付加してページ番号を表示します。 例えば、全 5 ページの Web ページをプリントするときは 3 ページ目に "3 / 5" を表示します。
      • ユーザ設定: あなたが使用したいテキストを入力してください。 特定の情報をプリントするために次のコードを含めることができます:
        • &PT: ページ番号と全ページ数 (例: "3 / 5")
        • &P: ページ番号
        • &D: 日付
        • &U: URL
        • &T: ページタイトル

ヒント: ページ設定での変更をプレビュー表示するには プリントプレビュー を使用してください。

スピードと効率を改善する

様々な種類のファイルを管理する

&brandShortName; は様々な種類のファイルを扱うことができます。 しかしながら、映像や音楽のような一部のファイルは、 それらを扱うことのできるプラグインや外部アプリケーションを必要とします。 そのファイルを開くのに必要なプラグインが &brandShortName; にインストールされていないときや、必要なアプリケーションが分からないときは、 ハードディスクにそれらのファイルを保存します。 ファイルを保存する場合、ダウンロードマネージャにその履歴を保持します。

プラグインは &brandShortName; を機能的に拡張し、その内部で実行されます。 Sun Java, Macromedia Flash そして RealNetworks RealPlayer などのプラグインはマルチメディアファイルを表示し、映像、アニメーション、 ゲームなどの小さなアプリケーションを実行します。 よく知られたプラグインは Mozilla Update の プラグインページで見つけることができます。

&brandShortName; 内でファイルを開けない場合は、 それらを開くために外部のアプリケーションを指定してください。例えば、MP3 ファイルを再生するためにお好みのメディアプレイヤーを指定します。

&brandShortName; で表示できないファイルは、ファイル読み込み時の動作設定 ダイアログでどのように扱うかを指定することができます。このダイアログを開くには、&pref.menuPath; を選択して コンテンツ パネルを開き、ファイルタイプ セクションの 管理... ボタンをクリックしてください。 各種のファイルに対する動作の変更や削除ができます。このダイアログについての詳しい情報は、ファイルタイプ&pref.plural; をご覧ください。

ファイルをクリックしても &brandShortName; がその扱いを知らない場合は、開く ダイアログが表示され、次の選択肢が提供されます:

  • アプリケーションで開く: システムで設定されたデフォルトのアプリケーションを使用してファイルを開きます。ドロップダウンリストの その他... を選択して異なるプログラムを指定することもできます。
  • ディスクに保存する: ファイルを保存する場所を選択してください。&pref.pluralCaps;の 一般 パネルで 次のフォルダに保存する を選択しているときは、 自動的にそこで指定されたフォルダに保存されます。 このオプションが選択されていないときは、保存先を尋ねられます。
  • 今後この種類のファイルは同様に処理する: これが選択されている場合、次回、同じ種類のファイルをダウンロードする際に 自動的に選択された動作を実行します。

ヒント: 今後この種類のファイルは同様に処理する で設定した内容は、ファイル読み込み時の動作設定 ダイアログで削除することができます。

&brandShortName; をデフォルト Web ブラウザにする

&brandShortName; は、インターネットやあなたのコンピュータ上の Web ページを表示します。Web ページを簡単に開くために &brandShortName; をデフォルト Web ブラウザにすることができます。

&brandShortName; をデフォルト Web ブラウザにするには、&pref.menuPath; を選択して 一般 パネル内の 今すぐ確認 ボタンをクリックしてください。

キャッシュ設定を変更する

&brandShortName; は、繰り返し訪れるページの複製をキャッシュに保存します。 これにより、そのようなページを訪れるたびにネットワークから取りなおす必要がなくなります。

キャッシュのサイズを設定するには:

  1. &pref.menuPath; を選択します。
  2. 詳細 パネルを開き ネットワーク タブをクリックします。
  3. 最大キャッシュサイズ フィールドに数値を入力してキャッシュサイズを指定してください。 デフォルトサイズは 50MB です。

重要: キャッシュサイズを大きくするほど多くのデータをすばやく取ってくることができますが、 より多くのハードディスク領域が使用されます。

&brandShortName; を終了するとキャッシュのメンテナンスが行われます。 メンテナンスにあなたが望むよりも長く時間がかかるときは、 ディスクキャッシュのサイズを小さくしてみてください。
