タブブラウズ

タブブラウズは、1 つの &brandShortName; ウインドウ内に各々の Web ページを表示する複数のタブを開きます。 あなたが現在のページを読んでいる間にリンク先を新規タブに開き、読み込ませておくことができます。 必ずしもタブブラウズを行う必要はありませんが、タブブラウズが Web を閲覧するのに早く、良い方法であることに気付くでしょう。

このセクションでは:
  • タブの作成と読み込み
  • ウインドウ内のタブを移動する
  • タブを閉じる
  • タブブラウズをカスタマイズする
  • ヒントと小技

タブの作成と読み込み

新規タブを開くには、&accelKey;+T を押下するか ファイル > 新規タブ を選択、またはタブバーの空いているところをダブルクリックしてください。

リンクをタブに読み込むには、以下のいずれかの操作をしてください:

  • リンクを中クリックする。 (マウスにホイールが付いているときは、ホイールのクリックと同じです)
  • リンクをドラッグし、タブバーの空いているところへドロップする。 (Web ページを 1 つだけしか開いていないときは、タブバーは隠れています。 タブブラウズをカスタマイズする を参照してください)
  • リンクをドラッグし、それを開きたいタブにドロップする。
  • リンクを右クリック&ctrlKey; を押下しながらクリックし、コンテキストメニューから リンクを新規タブで開く を選択する。

ロケーションバー内の URL を新規タブに開くには、 URL を入力し &altKey;+&enterKey; を押下してください。

ウインドウ内のタブを移動する

タブは開かれた順に表示されるため、いつもあなたの希望どおりに並ぶとは限りません。 &brandShortName; ウインドウ内でタブを別の場所へ移動するには マウスを使用してタブをドラッグするだけです。 タブをドラッグしている間、移動先を表す小さな矢印が表示されます。 また、キーボードショートカット を使用してウインドウ内のタブを移動することもできます。

タブを閉じ、復元する

現在のタブを閉じるには &accelKey;+W を押下するか、 タブを閉じるボタンをクリック、タブを中クリック、または ファイル > タブを閉じる を選択してください。 現在のタブを除いて他のタブをすべて閉じるには、タブを右クリック&ctrlKey; を押下しながらクリックして 他のタブをすべて閉じる を選択してください。

&brandShortName; のウインドウは最近閉じたタブの一覧を保持しています。 タブを復元するには、履歴 > 最近閉じたタブ から選択してください。すべてのタブを復元するには、履歴 > 最近閉じたタブ > タブですべて開く を選択してください。 &accelKey;+&shiftKey;+T を押下すると、個々のタブを過去に遡って開きます。

タブブラウズをカスタマイズする

タブブラウズ&pref.plural;を変更するには、&pref.menuPath; を選択して タブ パネルを開いてください。

ヒントと小技

ブックマークとタブブラウズ
タブグループをブックマークに追加するには:
  1. &accelKey;+&shiftKey;+D を押下するか ブックマーク > すべてのタブをブックマーク を選択して現在のウインドウ内のすべてのタブを新しいブックマークフォルダに保存します。
  2. ブックマークをタブグループで開くには、ブックマークフォルダのメニューを開いて タブで開く を選択してください。 またはブックマークフォルダを中クリックしてその内容をタブで開くことができます。 これらのブックマークは新規タブには開かれず、現在のタブ上に開かれます。
タブグループをホームページにする
1 つの Web ページのみをホームページとする代わりに、 タブグループをホームページにすることができます。 お気に入りの Web サイトを複数選択して 1 つのウインドウにタブで開いてください。 &pref.menuPath; を選択して 一般 パネル を開きます。起動 セクションの 現在のタブグループを使用 ボタンを押下してください。 ホームボタンを押下すると、お気に入りの複数の Web サイトが 1 回クリックするだけで読み込まれます。
新規タブツールバーボタンを追加する
ツールバーをカスタマイズ してツールバーに 新規タブボタン を追加することによって、簡単に新規タブを開くことができます。
キーボードショートカット
&brandShortName; にはタブブラウズに関する多くのショートカットが割り当てられています。 すべての一覧は キーボードショートカット をご覧ください。
