&brandShortName;

&brandFullName; について

&brandShortName; についての情報を表示します。 このダイアログボックスには、作者や貢献者の短い一覧と &brandShortName; のバージョン番号を表示します。

環境設定...

&pref.pluralCaps;ウインドウ を表示します。 &brandShortName; の多くの&pref.plural;を変更することができます。

&brandShortName; を終了

すべての &brandShortName; ウインドウを閉じてダウンロードを中止し、&brandShortName; を完全に終了します。複数のタブやウインドウを開いているときにこのメニュー項目を選択してしまったときは、 予期せず終了しないように確認のダイアログが表示されます。 この確認を表示しないようにするには、&pref.menuPath;の タブ パネル内にある 同時に複数のタブを閉じるときは確認する のチェックを外してください。 また、ファイルをダウンロード中である場合も確認のダイアログを表示します。

ファイル

新規ウインドウ

新しい &brandShortName; ウインドウを開きます。

新規タブ

現在の &brandShortName; ウインドウに新規タブを開きます。

URL を開く...

ロケーションバーにフォーカスを移動します。そこに訪問先の URL を入力してください。

ファイルを開く...

ファイルを開くダイアログを表示します。ローカルディスク、またはネットワーク上のファイルを選択してください。 &brandShortName; は HTML/XML ファイル、画像ファイル、テキストファイルなど多くの種類のファイルを表示することができます。

ウインドウを閉じる

現在のウインドウを閉じます。

タブを閉じる

現在のタブを閉じて、右のタブに移動します。 このメニュー項目は 1 つ以上のタブを開いているときのみ表示します。

別名でページを保存...

閲覧中のページを保存します。 ページ全体(画像も含む)、現在の HTML ページのみ、 またはテキストのみから選択して保存することができます。

ページの URL をメールで送信...

デフォルトメールクライアントでメール作成ウインドウを開き、 現在のページへのリンクをメッセージとして送信することができます。

ページ設定...

余白、ヘッダ、フッタなどのプリント設定と、 プリント方向を指定するためのページ設定ダイアログを表示します。

プリントプレビュー

現在のドキュメントがプリントされた場合の体裁を表示します。 ページ設定ダイアログでの設定とページ作者の意図が反映されます。

プリント...

プリント部数などを指定できるプリントダイアログを表示します。 OK をクリックするとページをプリントします。

設定とデータを読み込む...

設定移行アシスタントダイアログを開きます。&pref.plural;、ブックマーク、履歴、 パスワード、その他のデータを Microsoft Internet Explorer, Netscape, Mozilla または Opera から移行することができます。

オフライン作業

オンラインモードとオフラインモードを切り替えます。 オフラインモードでは、インターネットに接続されていなくても、 あなたが前に訪れた Web ページを見ることができます。

終了 終了

すべての &brandShortName; ウインドウを閉じてダウンロードを中止し、&brandShortName; を完全に終了します。複数のタブやウインドウを開いているときにこのメニュー項目を選択してしまったときは、 予期せず終了しないように確認のダイアログが表示されます。 この確認ダイアログを表示しないようにするには、&pref.menuPath; の タブ パネル内にある 同時に複数のタブを閉じるときは確認する のチェックを外してください。 また、ファイルをダウンロード中である場合も確認のダイアログを表示します。

編集

取り消す

テキストフィールドでの最後の編集を元に戻します。 取り消すコマンドを取り消したいときは、やり直すコマンドを使用してください。

やり直す

最後に元に戻したコマンドをやり直します。

カット

選択されたテキストをテキストフィールドから切り取り、クリップボードに保存します。 ペーストコマンドを使用してテキストを他の場所に貼り付けてください。

コピー

選択されたテキストをクリップボードにコピーします。 ペーストコマンドを使用してテキストをテキストフィールドに貼り付けてください。

ペースト

クリップボードに保存されたテキストをテキストフィールドに貼り付けます。

削除

選択されたテキストをテキストフィールドから削除します。

すべてを選択

ページ上のすべてのテキストと他の項目を選択します。

このページの検索...

ブラウザウインドウの下部に検索ツールバーを表示します。 検索するテキストをツールバーに入力すると、その度にページ内を自動的に検索し、 入力されたテキストで最初に該当するものを選択します。 次を検索 または 前を検索 をクリックすると、ページ内の別の位置のテキストを検索します。 すべて強調表示 をクリックすると、ページ内のすべての該当するテキストを強調します。

次を検索

このページを検索... コマンドで検索されたテキストの次に出現するものを検索します。

&pref.pluralCaps;

&pref.pluralCaps;ウインドウ を表示します。 &brandShortName; の多くの&pref.plural;を変更することができます。

表示

ツールバー

ナビゲーションツールバー

標準のナビゲーションボタン (戻る、進む、中止など) を持ったナビゲーションツールバーの表示状態を切り替えます。

ブックマークツールバー

様々なサイトのブックマークを含むブックマークツールバーの表示状態を切り替えます。

カスタマイズ...

ツールバーカスタマイズダイアログを表示します。 ツールバーボタンをツールバー上の配置したい位置にドラッグ＆ドロップしてください。 このダイアログは、ナビゲーションツールバー上で右クリック &ctrlKey; を押下しながらクリックし、カスタマイズ... を選択して表示することもできます。

ステータスバー

ウインドウ最下部に位置するステータスバーの表示状態を切り替えます。 ステータスバーは閲覧中のページに関して役に立つ情報を表示しますが、 すべての人に必要とされるものではないでしょう。

サイドバー

サイドバーはブックマークやページの表示履歴を表示します。 右上隅の×をクリックしてサイドバーを常に閉じておくことができます。

ブックマーク

いつでもブックマークを参照することができるように、ブックマークサイドバーを表示します。 あなたがブックマークをよく利用するなら、とても役に立つでしょう。

履歴

訪れたサイトの表示履歴を保持している履歴サイドバーを表示します。 履歴の保存日数を変更するには、&pref.menuPath; を選択して プライバシー パネルを選択してください。

中止

現在のページの読み込みを中止します。 これは、ツールバー上の 中止 ボタンをクリックするのと同じことです。

更新

閲覧中のページの最新版を読み込みます。 これは、ツールバー上の 更新 ボタンをクリックするのと同じことです。

文字サイズ

Web ページ上のテキストがあまりに小さかったり大きかったりする場合、より見やすいサイズに変更することができます。

大きく

Web ページ上のテキストを 1 サイズ大きく表示します。マウスでも同じことができます。 (マウスショートカット をご覧ください)

小さく

Web ページ上のテキストを 1 サイズ小さく表示します。マウスでも同じことができます。 (マウスショートカット をご覧ください)

標準サイズ

Web ページ上のテキストを標準のサイズで表示します。

スタイルシート

現在の Web ページのスタイルシートを選択することができます。 ページ作者が別のスタイルシートを指定していなければ、“標準スタイルシート” が選択されています。

スタイルシートを使用しない

ページからスタイルシートの書式を取り除きます。

標準スタイルシート

ページ作者に指定されたスタイルシートでページを表示します。

ページ作者が追加のスタイルシートを指定しているときは、 ここに選択できるスタイルシートの一覧を表示します。

文字エンコーディング

Web ページの文字エンコーディングを手動で変更することができます。 通常、&brandShortName; は文字エンコーディングを自動判別します。

ページのソース

現在のドキュメントのソースコードを表示します。

フルスクリーン

ウインドウを全画面で表示します。ナビゲーションツールバーを表示していないときは、 いつでも F11 を押下してウインドウを元に戻すことができます。

履歴

履歴 メニューは、最近訪れた Web サイトと閉じたタブの一覧を含んでいます。

戻る

訪れたページを 1 つ前のページへ戻ります。ツールバー上の 戻る ボタンのクリックと同じ動作です。

進む

訪れたページを 1 つ次のページへ進みます。ツールバー上の 進む ボタンのクリックと同じ動作です。

ホーム

ホームページを表示します。ツールバー上の ホーム ボタンのクリックと同じ動作です。ホームページを変更するには、 &pref.menuPath; を選択して 一般 パネルを開いてください。 空白ページ、現在のページ(またはタブグループ)、ブックマークから選択、 または直接 URL を入力して指定することができます。

最近閉じたタブ

一覧から選択することにより、現在のウインドウで最近閉じたタブを復元します。 詳しくは、タブを閉じ、復元する トピックをご覧ください。

サイドバーに表示

履歴サイドバーを表示します。履歴サイドバーは訪れたサイトの表示履歴を保持しています。 履歴の保存日数を変更するには、&pref.menuPath; を選択して プライバシー パネルを選択してください。

ブックマーク

ブックマーク メニューは保存されたブックマークをすべて含んでいます。

このページをブックマーク...

ブックマークに現在のページを追加します。 ブックマークのタイトルと作成先を指定するためのダイアログが表示されます。

このページを購読...

現在のサイトによって提供されている フィード のプレビューを表示します。プレビューページでは、ライブブックマーク やコンピュータ上のフィードリーダー、Web サービスを利用してフィードを購読することができます。&pref.menuPath; の アプリケーション パネルで、常にライブブックマークまたはフィードリーダーを使用して購読するよう設定しているときは、 プレビューページは表示されません。

すべてのタブをブックマーク...

新しいブックマークフォルダに現在のウインドウ内のすべてのタブを追加します。 フォルダ名と作成先を指定するためのダイアログが表示されます。

ブックマークの管理...

ブックマークの管理ウインドウを表示します。 ブックマークの並べ替え、名前の変更、プロパティの変更に加えて、 これらの追加、削除、移動ができます。

ツール

Web 検索

Web 検索バーにフォーカスを移動します。 Web 上の検索したい単語を入力してください。

ダウンロード

ダウンロードマネージャを開きます。 進行中のダウンロードと完了したものを確認することができます。

アドオン

アドオンマネージャを開きます。拡張機能とテーマの確認、インストール、設定、更新、アンインストールを行うことができます。 詳しくは、アドオン トピックをご覧ください。

エラーコンソール

JavaScript コードの問題を追跡するエラーコンソールを開きます。 JavaScript は、Web ページを構築するために一般的に用いられるスクリプト言語です。 プログラマは Web ページをより対話的にするために JavaScript を利用します。 例えば、状況に応じたフォームやボタンの検証などによく利用されます。

ページの情報

ページの情報ダイアログは、ドキュメントタイプ、文字エンコーディング、 サイズ、セキュリティ情報などの現在の Web ページに関する広範囲な情報を表示します。 また、ページ上で使用されているメディアとリンクの一覧を表示します。

プライバシー情報の消去 ...

&pref.menuPath; の プライバシー パネル内にあるプライバシー情報の消去ダイアログで選択された項目を消去します。 デフォルトでは確認のダイアログが表示されます。

&pref.pluralCaps;

&pref.pluralCaps;ウインドウ を開きます。 &brandShortName; の多くの&pref.plural;を変更することができます。

ウインドウ

しまう

現在のウインドウをしまいます。

拡大／縮小

現在のウインドウをズーム、または元に戻します。

(ウインドウを開く)

そのタイトルによって開かれたすべてのウインドウを表示します。

ヘルプ

&brandShortName; ヘルプ &brandShortName; ヘルプ

&brandShortName; ヘルプウインドウを開きます。 Web を閲覧するために役に立つ情報があります。

Internet Explorer ユーザのために

Internet Explorer ユーザが &brandShortName; へ移行するための情報をヘルプウインドウに表示します。

リリースノート

&brandShortName; のインストールとアンインストール、設定についての情報、 他の重要な情報と最新の注意事項を提供しています。 このリリースノートを読むにはインターネットに接続している必要があります。

不具合のあるサイトを報告...

&brandShortName; で正しく表示できない、または動作しない Web サイトを &brandShortName; の開発者に報告します。

偽装サイトを報告... / 誤警告を報告...

偽装サイトを報告... は、偽装または "フィッシング詐欺" と思われるサイトを報告することができます。 フィッシング詐欺サイトは、あなたを騙して個人情報(ユーザ名とパスワード、クレジットカード情報など) を盗むために、銀行やオンラインショッピング、オークションサイトなどのあなたの信用するサイトを装います。

訪れたサイトが "フィッシング詐欺" の疑いがあると表示された場合は、誤って "フィッシング詐欺" サイトであるとマークされた正当なサイトを報告するために、 メニュー項目が "誤警告を報告..." に置き換えられます。

&brandShortName; のフィッシング防御は Google のシステムを使用しており、 報告した情報は Google に送信され、この機能の向上に役立てられます。 あなたの報告は Google のプライバシーポリシーに応じて匿名になります。 報告するにはインターネットに接続する必要があります。

ソフトウェアの更新を確認...

&brandShortName; の更新を確認するダイアログを表示し、更新されているときは、それをダウンロードするかどうか尋ねられます。 ダウンロード後、最新版をインストールするために &brandShortName; を再起動するかどうか尋ねられます。 最新版のダウンロード中、またはインストールの準備ができている場合は、このメニューの項目名が変わります。
